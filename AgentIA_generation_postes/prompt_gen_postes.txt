Créez {{nombre_postes}} postes distincts pour le métier {{metier}} en suivant les instructions ci-dessous. Chaque poste doit inclure les informations suivantes :

- Nom de l'entreprise : Générez un nom d'entreprise original et unique.
- Titre du poste : Créez un titre de poste spécifique et pertinent pour le métier.
- Description du poste : Rédigez une description détaillée et réaliste, en évitant les généralités.
- Responsabilités : Listez des responsabilités spécifiques au poste, en étant précis et exhaustif.
- Compétences requises : Énumérez les compétences nécessaires, en détaillant chaque compétence.
- Localisation : Indiquez la localisation du poste en Europe.
- Niveau d'expérience : Spécifiez le niveau d'expérience requis (Junior, Intermédiaire ou Senior).
- Éducation requise : Précisez les qualifications éducatives nécessaires pour le poste.

Formatez chaque poste dans le format JSON suivant :

```json
{
  "company_name": "",
  "job_title": "",
  "job_description": "",
  "responsibilities": [],
  "skills_required": [],
  "location": "",
  "experience_level": "",
  "education_required": ""
}
```

Effectuez un contrôle qualité pour :

- Vérifier que chaque poste est unique et qu'il n'y a pas de duplicatas.
- S'assurer que les descriptions, responsabilités et compétences sont spécifiques et non génériques.
- Vérifier que vous avez bien créé la quantité de postes demandés.

Rassemblez tous les JSON des postes dans une liste sous la clé "postes", par exemple :

```json
{
  "postes": [/* liste des postes */]
}
```
